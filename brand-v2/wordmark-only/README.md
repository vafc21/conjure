# Wordmark-Only Option

**Rationale:** Following the Linear/Notion model — a text-only brand mark with no icon.
"Conjure" set in Inter SemiBold with tight tracking and a clay-red period at the end. The
period is a deliberate typographic detail: it reads as a command terminator (terminal
prompt `$` energy), a full-stop on the "conjuring" (the app is ready), and a visual anchor
that makes the wordmark distinctive without an icon.

For app header usage where no lockup is needed, use the wordmark alone. For favicon/16×16
contexts, use the C-monogram favicon from Direction C or the abstract mark from Direction A.
