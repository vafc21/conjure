# Direction B — The Sketch-to-Code Stroke

**Rationale:** A purely abstract mark that tells the Conjure story in a single line. The
left side is a gentle undulating curve — the loose, organic feel of a pencil sketch. At the
center it snaps into precise geometric angles, forming a `>` chevron that reads as code, a
"next step" prompt, or a play button. No window, no frame — just the transformation itself.

The chevron on the right is a universal symbol in developer tools: the terminal prompt, the
forward arrow, the "run" button. The mark works left-to-right as a narrative: input →
transform → output.
