# Direction C — The C Window Monogram

**Rationale:** A monogram logo that works both as a letter mark (the "C" in Conjure) and
as a window frame waiting for content. The C is built as three sides of a rounded app
window — the open side on the right represents possibility, the thing being built. Inside
that opening, a clay-red chevron acts as a cursor pointing forward: the next pixel, the
next line of code, the next frame of the app materializing.

At small sizes the chevron reads as a simple accent mark, but at full size the dual
C + window + cursor meaning becomes clear.
